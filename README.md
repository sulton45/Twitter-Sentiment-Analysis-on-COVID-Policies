Masyarakat di Indonesia sering memberikan komentar terhadap kebijakan pemerintah dalam menangani masalah virus COVID-19 melalui media sosial, salah satunya adalah Twitter. Komentar masyarakat baik secara positif, netral maupun negatif terhadap kebijakan pemerintah dapat dianalisis dengan menggunakan analisis sentimen. Dalam analisis sentimen ini akan menggunakan dua metode yaitu Support Vector Machine dan Naive Bayes dengan Laplace Estimator. Penelitian ini bertujuan untuk mendapatkan perbandingan tingkat akurasi dari hasil kedua metode tersebut.
